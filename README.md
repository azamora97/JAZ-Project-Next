# 🚀 Proyecto JAZ-DevBoard

📘 Proyecto del curso de React Avanzado

## 📦 Instalación y ejecución

### 1. Clonar el repositorio

git clone https://github.com/azamora97/JAZ-Project-Next

Ingresar a la carpeta: cd JAZ-Project-Next

Instalar las dependencias: npm install

Ejecutar el proyecto en desarrollo: npm run dev

Ingresar a la ruta: http://localhost:3000/

📌 Ruta principal del módulo de proyectos: http://localhost:3000/projects

### 2. 🧪 Pruebas unitarias

npm run test

### 3. 🔍 Calidad de código (Lint)

npm run lint
